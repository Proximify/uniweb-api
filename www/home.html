Uniweb API Client Dashboard

This dashboard offers a list of links to API queries that demonstrate common operations. The source code of the queries can be found at the queries sub-folder. Please read the individual example source code to know more about each query.

Example API queries

Query Description
Fetch Profile data Generate an HTML page using JSON data fetched by the API client.
Get the members of a unit Choose one 'University' unit and request all members in it.
Fetch CV data Request information about sections, fields * and the valid options for drop-down fields.
Add items to Profile Add two research interests to the profile of a user. Note: Requires API write mode.
Add items to CV Add new items into a CV section. Note: Requires API write mode.
Query with filter;
Get interests		 1. Query member data with the filter "loginName" (returns an array).
2. Iterate over the interests.
Query by ID;
Get interests		 1. Query by ID (returns a single element)
2. Iterate over the interests.
Edit member info Edit some of the information on the membership section. Note: Requires API write mode.

Note that if there is already an item with data in the section, then only the field values that
you send will modify existing values. For example,if you don't send a middle name, and the user
had set a middle name, then the existing middle name will be unchanged.
Edit the "user profile" section in the CV There is a CV section named "user profile". IT is NOT the Uniweb Profile of a member. It is a CCV section.
Requests publications Like example 1 but it selects publications instead of interests.
The query assumes an academic unit named Engineering.
Add/edit the values of "select" fields Using fields that offer a dropdown of options to the users.
Pass the options as text instead of retrieving the IDs of the options first.
Change the profile picture (URL) Change the profile picture of a user with an image given as a URL.
Change the profile picture (file) Change the profile picture of a user with an image given as a local file path
